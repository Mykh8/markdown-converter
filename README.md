# Markdown to HTML converter

A simple command-line tool that converts basic Markdown syntax to HTML

## Features
- Headers (h1-h6)
- Bold text (`**text** or __text__`)
- Italic text (`*text* or _text_`)
- Inline code (`` `code` ``)
- Blockquotes (`> quote`)
- Paragraphs

## Usage
``` bash
java MarkdownConverter input.md output.html
```
